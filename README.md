# Sistema para Gestao de ONG

Projeto desenvolvido para a disciplina de Programacao Orientada a Objetos.

## Objetivo

Simular um sistema basico de gestao para uma ONG, aplicando conceitos de Programacao Orientada a Objetos em Java.

## Conceitos utilizados

* Encapsulamento
* Heranca
* Composicao
* Sobrescrita de metodos (@Override)

## Classes do projeto

* MembroONG
* Voluntario
* Doador
* ProjetoSocial
* SistemaMain

## Funcionalidades

* Cadastro de voluntarios
* Cadastro de doadores
* Associacao de lider a um projeto social
* Exibicao de resumos dos membros
* Inicializacao de um projeto social
